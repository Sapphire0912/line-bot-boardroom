## 專案名稱：Line Bot 留言板

全端開發技術練習，Line Bot 留言板功能實作。採用 Next.js 全端框架技術，學習 Line Message API 串接、前端設計、伺服器邏輯實作以及資料庫操作。

## 使用技術

### 前端

- **Next.js**
- **React.js**
- **Tailwind CSS**
- **Context API**
- **RWD**
- **JWT**

### 後端

- **Node.js**
- **Javascript/Typescript**
- **MongoDB**

### 其他

- **Line Message API**
- **Line Login**
- **Firebase**
- **Vercel**
